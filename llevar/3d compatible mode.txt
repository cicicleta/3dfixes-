II. Step 1 - How to export current Nvidia Profiles
==================================================

(Believe it or not Nvidia inspector doesn't export properly the profiles)

1. First go to "C:\ProgramData\NVIDIA Corporation\Drs" Folder.

2. We are interested in these 3 files:
  nvdrsdb0.bin  - DataBase0 binary file
  nvdrsdb1.bin  - DataBase0 binary file
  nvdrssel.bin  - DataBase Selector file between 0 and 1

3. Delete the   nvdrssel.bin  file. (It will get automatically generated on profile import)

4. Start   Geforce 3D Profile Manger  and hit [Export SLI Profiles]. Save the file.

5. Open the file and search for your DX11 Game entry.

6. Once found APPEND or REPLACE in the settings section this entry:  

    Setting ID_0x709adada = 0x07f58257 InternalSettingFlag=V0

===========================================
II. Step 2 - Explaining what the flag means
===========================================

 

    Setting ID_0x709adada = 0x07f58257 InternalSettingFlag=V0 

  
 Setting ID_0x709adada  - The "magical" 3D Compatibility Flag
  0x07f58257  - Value of the flag, which   controls how 3D Compatibility Mode WORKS  .
  InternalSettingFlag=V0  - This lets the driver knows this is a predefined value. If this is missing   3D Compat Mode will not kick in  .
Ex:  

    Setting ID_0x709adada = 0x07f58257 

 =========================================
II. Step 3 - Possible Values for the Flag
=========================================

Like I said above, the value controls how 3D mode works and mainly  
 what is 3D and what 2D  .

Values that I found interesting:
 

    From:
    Setting ID_0x709adada = 0x37f58240 InternalSettingFlag=V0
    // Used by Bioshock Infinite
     
    Commonly used:
    Setting ID_0x709adada = 0x37f58257 InternalSettingFlag=V0
    // Used by most of the games
     
    To:
    Setting ID_0x709adada = 0x37f5825a InternalSettingFlag=V0
    // My special value. See below the Example. 

 Finding the correct value is GAME dependent. Just try from the lower value and increase and see what you get. When one found stick with that;))



=========================================================================
III. Example: Getting the most of the CM mode in Dragon Age: Inquisition
=========================================================================

Screenshot taken in Discovery Mode so they can be seen better:

1.  

    Setting ID_0x709adada = 0x37f58240 InternalSettingFlag=V0 

 - Flat 2D. Nothing is actually rendered in 3D.

2.  

    Setting ID_0x709adada = 0x37f58547 InternalSettingFlag=V0 

 - Stereo 3D. Everything is rendered in 3D, including Menus + UI. Known for the atrocious jumping/broken/haloing UI.

3.  

    Setting ID_0x709adada = 0x37f5854a InternalSettingFlag=V0 

 - Stereo 3D. Game is rendered in 3D. Menus + UI is rendered in 2D so no more weird/broken/haloing UI.

V.HOW-TO EXPORT and IMPORT Nvidia Profiles using "Geforce 3D Profile Manager" 
==============================================================================
This video shows how to export/import profiles. It doesn't cover the whole aspect of finding the CM flag as above.


 Lo Principal: Dos Métodos
Método Moderno (Recomendado)
Usa 3DMigoto (versión reciente) como wrapper library. Es más avanzado pero muy flexible: puedes tweakear cualquier juego DX11 sin tocar perfiles de NVIDIA. A veces requiere configuración para que no crashee el juego.
Método Antiguo (Lo que explica la mayor parte del documento)
Edita manualmente los perfiles NVIDIA usando NVIDIA Inspector o Geforce 3D Profile Manager para habilitar Compatibility Mode (CM).

🔧 El Método Antiguo Paso a Paso
Paso 1: Exportar perfiles NVIDIA

Ve a C:\ProgramData\NVIDIA Corporation\Drs
Busca tres archivos:

nvdrsdb0.bin y nvdrsdb1.bin (bases de datos)
nvdrssel.bin (selector entre BD0 y BD1) — ELIMINA ESTE


Abre Geforce 3D Profile Manager → click en [Export SLI Profiles] → guarda el archivo
Abre el archivo exportado y busca tu juego DX11

Paso 2: El "Magic Flag"
Añade o reemplaza esta línea en la sección Settings de tu juego:
Setting ID_0x709adada = 0x07f58257 InternalSettingFlag=V0
Desglose:

ID_0x709adada = la bandera mágica que activa Compatibility Mode
0x07f58257 = el valor que controla cómo funciona el CM (qué es 3D y qué es 2D)
InternalSettingFlag=V0 = le dice al driver que es un valor predefinido (¡CRÍTICO! Sin esto no funciona)

Paso 3: Posibles Valores para el Flag
Diferentes valores dan resultados distintos:
ValorResultado0x37f58240Plano 2D — nada en 3D0x37f585473D total — todo (menus + UI) en 3D → pero la UI sale deformada/rota0x37f5854a3D + UI 2D — juego en 3D, menus/UI en 2D (sin artefactos)
Clave: El valor correcto depende del juego. Tienes que probar desde valores bajos e ir subiendo hasta encontrar el que funcione bien.
Paso 4: Importar de vuelta

Elimina nvdrssel.bin nuevamente
En Geforce 3D Profile Manager → [Import SLI Profiles]
Lanza el juego — Compatibility Mode debería estar activado


⚠️ Advertencia Importante
ANTES de hacer esto, revisa el HelixMod Blog. Ya han arreglado cientos de juegos con stereo 3D propio, así que es probable que tu juego ya tenga una solución mejor. También puedes usar 3D Fix Manager para instalar fixes automáticos si existen.

🎯 En Resumen
Es un método manual para habilitar modo 3D en juegos DX11 que no lo tienen nativamente. Editando un flag en los perfiles NVIDIA, le dices al driver cómo renderizar el contenido en stereo 3D — pero hay que ajustar el valor correcto para cada juego para evitar que la UI salga hecha un desastre.
